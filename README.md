# sfondeur.dev

Personal portfolio built with React, TypeScript, and Vite.

## Tech Stack

- Typescript
- React 19
- Vite + SWC
- MUI v7
- React Router v7

## Development

```bash
npm install
npm run dev
```

## Scripts

| Command          | Description          |
| ---------------- | -------------------- |
| `npm run dev`    | Start dev server     |
| `npm run build`  | Production build     |
| `npm run lint`   | Run ESLint           |
| `npm run format` | Format with Prettier |
